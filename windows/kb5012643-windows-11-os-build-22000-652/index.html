[版本更新] 快速回顧 微軟發布 KB5012643 (Windows 11 作業系統組建 22000.652)

發表於 2022-05-01 更新於 2023-09-01 分類於 windows 文章字數： 838 所需閱讀時間 ≈ 2 分鐘

前言

4 月 25 日，微軟針對 Windows 11 釋出 KB5012643 (作業系統組建 22000.652) 的更新。

更新前內容概覽

有興趣的伙伴，更新前可以先讀一下 2022 年 4 月 25 日 - KB5012643 (作業系統組建 22000.652) 預覽 裡的說明。

有 4 個更新重點，我閱讀之後的想法如下。

  1. 更新可能會導致部分截斷視訊字幕的問題：這個問題我沒有遇到。
  2. 更新視訊字幕不正確對齊的問題：這個問題我沒有遇到。
  3. 在工作列上的天氣圖標上方顯示溫度：一直都有顯示，之前我是有注意到沒有登入帳號，似乎就不給顯示，反正我是都會登入，後來都有正常顯示。其實，我在 Windows 10 上頭，還可以看到美股與台股的漲跌，以及濕度顯示，雖然我沒有投資股票，但這個地方可以是對當日經濟活動的小觀察，我個人認為能夠顯示這個是滿不錯的，只不過，我沒有去摸過它的設定，所以在 Windows 11 上面，只看過溫度，其他資訊都沒有讀過，目前也還不清楚怎麼處理。
  4. 更新導致您無法在最大化的應用程式視窗上使用最小化、最大化及關閉按鈕的問題：這個問題我也沒有遇到過。

暫時不更新

前不久因為要跟著面對疫情，而讓手邊跑 Windows 11 的機器忙碌一陣子，現在又回到休閒狀態(真好命)，所以，我認為已經不用急著把新補丁隨時打上去。

再者，我觀察網路上網友們的更新狀況，再三思考後決定不更新，主要就是這次更新重點提到的狀況，我都沒有遇到過，以前我會想著要防微杜漸，但是在使用 Windows Update 的經驗裡，我常常是更新之後問題更多。

易言之，常常是不更新還好，一更新就出狀況。

後記

我還是會持續注意 Windows 11 的更新補丁，但，我不見得會隨著每一次升級有所動作。只是，我所使用的電腦，非工作用電腦，裡面也沒有什麼太重要的資料，更不會拿來上網做一些消費的動作，因此，安全性的部分，已被我大大忽略。

如果是常用的主力電腦，我個人認為，觀察一下，沒有太大災情，就應該要更新比較好。
