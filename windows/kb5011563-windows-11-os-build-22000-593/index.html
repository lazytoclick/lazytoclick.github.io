[版本更新] 快速回顧 微軟發布 KB5011563 (Windows 11 作業系統組建 22000.593)

發表於 2022-04-08 更新於 2023-09-01 分類於 windows 文章字數： 566 所需閱讀時間 ≈ 1 分鐘

前言

我有注意到 4 月份的 Windows 11 品質更新，已經釋出：KB5011563 (作業系統組建 22000.593)。

更新前內容概覽

我先大致讀了一下 2022 年 3 月 28 日 —KB5011563 (OS 版本 22000.593) 預覽 裡的說明。

更新重點有 3 點，我閱讀之後的想法如下。

  1. 「最多同時顯示三個高優先順序的 Toast 通知」，坦白說…，我不是很清楚這個功能帶給我什麼樣的便利。
  2. 延續著前面的 KB5011493 (Windows 11 作業系統組建 22000.556) 更新，持續在修復 Microsoft OneDrive 的 Bug。
  3. 最後一個重點是「當您搜尋設定小工具時，會返回對應的頁面。」，感覺我好像跟 Windows 11 實在太不熟，有沒有「返回對應的頁面」，對我而言，也沒有很重要。

暫時不更新

從上個月底到現在，除了我沒空更新外，偶爾想到時，我總認為我暫時不會想把這支補丁打上去。

後記

我還是會持續注意 Windows 11 的更新補丁，但，我不見得會隨著每一次升級有所動作。

只是，我所使用的電腦，非工作用電腦，裡面也沒有什麼太重要的資料，更不會拿來上網做一些消費的動作，因此，安全性的部分，已被我大大忽略。

如果是常用的主力電腦，我個人認為，觀察一下，沒有太大災情，就應該要更新比較好。
