[版本更新] 快速回顧 微軟發布 KB5014697 (Windows 11 作業系統組建 22000.739)

發表於 2022-06-18 更新於 2023-08-03 分類於 windows 文章字數： 737 所需閱讀時間 ≈ 1 分鐘

前言

6 月 14 日，微軟針對 Windows 11 釋出 KB5014697 (作業系統組建 22000.739) 的更新。

更新前內容概覽

  1. 官方說明文件：2022 年 6 月 14 日 - (作業系統組建 22000.739) KB5014697，說明文件提到這支更新的重點(如下)。

    解決Windows作業系統的安全性問題。

  2. 官方文件也提到改進的內容是「此更新包含內部作業系統功能的各項安全性改良功能。」

災情與駭客

  1. 災情：因為是安全性更新，通常我不會想太多，也不會等太多天才更新，所以，當系統跳出更新訊息不久，我就把補丁打上。使用上沒有問題，所以，我就沒有想太多。然而，在撰寫本文之前，特意搜尋一下，發現這支補丁是有災情的，詳可見：微軟坦言 Windows 更新又出包！一按鍵網路秒斷線　所有版本都遭殃 - 自由電子報 3C科技、Latest Windows Update breaks Wi-Fi hotspot feature for some PCs | Windows Central。
  2. 駭客入侵：這一支補丁，網路上有風聲，主要是把安全性的洞口補起來，詳可見：Urgent! Don’t wait to install the June 2022 Windows update - it will protect you from Chinese hackers - Tips 2 Fix。
  3. 魚與熊掌：這樣的比喻有點怪怪的，不過，這支補丁似乎是要讓使用者選擇，看是要讓系統開啟熱點就自己掛點，還是要讓駭客入侵。

後記

雖然我是更新之後，才知道有災情，但，也由於我的機器上從來不會開「行動熱點」，所以，應該影響是不會太大。

截至目前，使用約 2 天，倒沒有特別奇怪的狀況發生。
