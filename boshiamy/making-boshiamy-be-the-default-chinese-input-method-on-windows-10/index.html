在 Windows 10 將嘸蝦米設定為預設中文輸入法

發表於 2022-02-10 更新於 2024-03-17 分類於 boshiamy 文章字數： 754 所需閱讀時間 ≈ 1 分鐘

前言

我將設定大致分為 2 個方向：

  1. 設定嘸蝦米為預設且唯一的中文輸入法
  2. 保留其他中文輸入法，設定嘸蝦米為「預設輸入法」

設定嘸蝦米為預設且唯一的中文輸入法

  1. (↓)【開始】→【設定】，進入以下畫面。

  2. (↓)【時間與語言】→【語言】後，來到以下畫面。【慣用語言】裡，只留下【中文(繁體，台灣)】，並且點擊【中文(繁體，台灣)】裡的【選項】。

  3. (↓)移除微軟注音輸入法，只保留【嘸蝦米輸入法】。

  4. (↓)回到桌面的工作列，右下角找到【英】或【中】，用滑鼠右鍵點擊，會跳出嘸蝦米的設定選單，並且點一下選單上的【設定】。

  5. (↓)我想要一進系統以【英數模式】為預設輸入模式，保留【使用 Shift 切換「中文/英文」模式】，不要【繼承系統輸入法狀態】。

  6. 走完以上的設定，基本上 Windows 10 上就剩嘸蝦米輸入法，很多設定見人見智，以【使用 Shift 切換「中文/英文」模式】為例，我在不一樣的機器，就不一定會保留，例如：我在 Windows 11 上面設定嘸蝦米的筆記，我就做出不一樣的設定，主要原因是電腦不是我專用，而我老婆同為蝦米族，但她的使用習慣有微微的差異。

保留其他中文輸入法，設定嘸蝦米為「預設輸入法」

  1. 嘸蝦米輸入法已經是我在 macOS、Linux 上唯一的輸入法，反而是在 Windows 上面，我還保留著微軟注音輸入法，主要是考量到可能會有其他的使用者，偶爾會借用我的電腦。

  2. 如果要跟其他中文輸入法共存，又希望嘸蝦米輸入法是預設的輸入法，關鍵在【設定】裡的【覆寫預設輸入法】，建議可以透過【設定】介面裡的【尋找設定】，直接用關鍵字把【覆寫預設輸入法】找出來，然後做進一步的設定，就可以達成。
