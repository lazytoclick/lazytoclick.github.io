安裝 Fcitx5 for Android(小企鵝輸入法5)與嘸蝦米

發表於 2024-04-28 更新於 2024-09-06 分類於 boshiamy 文章字數： 945 所需閱讀時間 ≈ 2 分鐘

前言

老早以前，我就注意到 Fcitx5 for Android 這支 App。

雖然，我曾經抱著好奇的心態下載與使用 Fcitx5 for Android 這支 App，但是，可能是我已經有官方的嘸蝦米輸入法 App，所以，啟動 Fcitx5 for Android 後，並沒有仔細研究如何把嘸蝦米輸入法表格掛載上去。一個機緣之下，發現已經有網友成功處理完成，我參考網友的說明後，發現設定的要點與程序，真的是「藏得有點深」，所以，決定以螢幕截圖的方式，紀錄自己走過的步驟。

安裝過程

安裝 Fcitx5 For Android 這支主程式

  1. 先跑一趟 Google Play 應用程式商店（商店傳送門：Fcitx5 for Android - Google Play 應用程式），把 Fcitx5 For Android 安裝進來。

  2. 安裝完成後，找到這支 App：「小企鵝輸入法5」。

掛載嘸蝦米輸入法

  1. 到掛載嘸蝦米輸入法表格相關檔案的設定前，要走不少步驟，所謂的「藏得有點深」，指的就是這個。



  2. 走到這裡，有 2 種方法可以將嘸蝦米輸入法表格相關檔案掛進來，我選用的方法是跑一趟 GitHub 把 boshiamy.conf.in 與 boshiamy.txt 載到手機裡面，再分別掛載上去。


  3. 回到一開始的設定畫面，再從[輸入法]進去，把「Boshiamy」新增進來。



  4. 承上圖，確認「已新增 Boshiamy」後，我會把[Boshiamy]的順序往上拉。

切換輸入法至「小企鵝輸入法5」

  1. 切換輸入法至「小企鵝輸入法5」後，要再選擇[Boshiamy]，才算是真正切換到嘸蝦米輸入法。


  2. 最後，不免俗地試試看能否順利吃蝦。
