ubuntu 23.04 與嘸蝦米

發表於 2023-09-04 更新於 2024-10-20 分類於 boshiamy 文章字數： 439 所需閱讀時間 ≈ 1 分鐘

前言

來試試在 ubuntu 23.04 快速安裝 fcitx(4) 並啟用嘸蝦米。

安裝過程

Fcitx 在 ubuntu 23.04 並不是預設的中文輸入平台，不過，仍然可以在幾個步驟內就搞定。

  1. 一個指令，就讓相依的套件都安裝進來。

  2. 進入系統的「地區和語言」。

  3. 點進「管理安裝的語言」(Manage Installed Language)，把「鍵盤輸入法系統」設定為【Fcitx 4】。

  4. 重新啟動(Restart…)。

  5. 順利的話，重新進入系統，就會看到工作列上面，多出了一個鍵盤，而且已經把嘸蝦米安裝進來。

  6. 使用 Ctrl+Space(空白鍵)，切換到嘸蝦米時，畫面上會跳出被框框包圍住的藍色的「嘸蝦米」或「Boshiamy」字樣，同時，在右上角工作列那邊，原本的「鍵盤」圖示，也變成紅色的「嘸」。
