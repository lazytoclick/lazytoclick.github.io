來玩 ArchLabs Linux 2023.01.20

發表於 2023-01-25 更新於 2024-03-17 分類於 linux 文章字數： 928 所需閱讀時間 ≈ 2 分鐘

前言

ArchLabs 是一支 Arch Linux 的衍生發行版(從字面上就滿容易辨識的)，目前在 DistroWatch 上的「網頁參觀排名」是第 80 名。

快速回顧與感想

  1. 安裝與初始設定：我這次是在 VirtualBox 上安裝，從 ISO 檔進入系統後，可以發現，似乎只能選擇以 CLI 的模式來進行系統安裝。我也好久沒有在這樣的介面下進行安裝，也算是走一趟回憶之旅。

    (↓)還有更多的照片，可以點下面的相簿瀏覽一下。

  2. Fluxbox：之前，我曾經在使用 MX-Linux 時，考量過要不要使用 Fluxbox，今次，就在安裝 ArchLabs 時，決定裝來用用，也算是初體驗。同時，在我體驗後，有比較多的感受是介面清爽、精簡，而且執行速度相當有效率。

中文支援、設定與中文輸入法框架的使用心得

  1. 中文字體與 Fluxbox 的設定檔是中文支援的關鍵。

    (↓)在安裝的過程，我已經算是做好設定，但初次載入系統，卻看到這個「亂碼」按鍵。經驗上跟我說，這個應該是在安裝過程，沒有把中文字體安裝進來。

    (↓)中文字體，倒是好解決，下個指令就可以搞定。

    (↓)不過，請相信我，即使安裝好，也沒有辦法正確顯示，這個是因為還要改動 theme.cfg 這個檔案裡，有關字體的設定。

    (↓)為求快速，我直接套 Fluxbox 的 System Styles，而不使用 ArchLabs 的 User Styles，其實，也可以把 Fluxbox 的 System Styles 複製貼到 User Styles 就好，只是，我認為 System Styles 就很好看，即使中文支援還不到百分之百，都比處處都是「框框」來得容易使用個數百倍(吧！)。

  2. Fcitx 5：安裝與使用過 Arch Linux，以及為數不少的衍生發行版，已經可以說，服用 Fcitx5 這一帖，就能夠搞定中文輸入的需求。
