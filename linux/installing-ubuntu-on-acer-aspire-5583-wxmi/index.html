在 Acer Aspire 5583 WXMi 上面安裝 ubuntu

發表於 2008-05-31 更新於 2024-03-17 分類於 linux 文章字數： 610 所需閱讀時間 ≈ 1 分鐘

W-BLUE 就是太太在使用的王建民 19 勝紀念機… (( 即 Acer Aspire 5583 WXMi，更詳細的規格可再參考 Acer宏碁台灣官網。 ))

ubuntu 會安裝到 W-BLUE 也算是個偶然，在 Windows Vista SP1 表現不如預期的情況下，忍痛請它先行下場休息，並請歷年來最穩定的老大哥 Windows XP Home 上場，熱身時間較長，還得將最新的加持包 Service Pack 3 掛上去，費時費力，因此待整個安裝完成後，我立刻想到之前跟 ubuntu 這位小老弟愉快的合作經驗，而今次它仍舊讓人滿意。

有別於前次的 Install inside Windows 合作方式，今次我想要自行分割硬碟，好讓 ubuntu 順勢上場，也在此時下定決心將 acer 原本所設定的 8 GB 系統備份還原區砍下來。對於硬碟的規畫，我的設想是用於桌面的 Linux ，分割區不用特別要求，又逢 x86 機器的限制，因此只切出 root 與 Swap，就讓 ubuntu 上去。

應用程式的部分，我還沒有完全搞定，倒是先玩一下 Cheese，對 Logitech 的視訊表現感到相當滿意，附帶一提的是驅動程式的表現都正常，只剩 HPLIP Toolbox 還無法搜尋到 HP Photosmart C4385 (( PCMCIA 不在測試之列，因為我沒有相關的硬體可以玩看看！ ))。
