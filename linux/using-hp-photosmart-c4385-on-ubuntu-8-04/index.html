在 ubuntu 8.04 上使用 HP Photosmart C4385

發表於 2008-05-21 更新於 2024-03-17 分類於 linux 文章字數： 522 所需閱讀時間 ≈ 1 分鐘

HPLIP Toolbox 2.8.2 版已經可以順利抓到 HP Photosmart C4385 向空中所發出的訊號，再透過幾個簡單的步驟就可以讓 ubuntu 8.04 順利地與其串連在一塊兒。我試著列印，完全沒有問題。

只有列印功能是不是讓人感到滿意呢？對我而言，已經解決很多問題，可是不死心地將 XSane Image Scanner 挖出來，程式啟動後，除了順利抓取 Webcam，連 HP Photosmart C4385 也身在其中，緊接著把方才列印出來的放到平台上嘗試，哇！真的、真的可以掃描耶！萬歲！怪哉！HPLIP 的官方網站的相關頁面：Photosmart c4380 Series，實在讓人讀不出其支援 Wireless 串接的相關說明。

日前所述，那 5% 需要請 Windows Vista 出面方能解決的問題，現在已不復存在，也就是說可以乾脆把 Windows Vista 殺掉，建構完全的烏托邦嚕！ (( 實際上我沒有那麼衝動，將現成的 Windows Vista 擱著也不會讓我坐立難安，既然如此又何必苦苦相逼呢！？況且，我還把幾個自己喜歡在 Windows 上面使用的字型拿到 ubuntu 來用。 ))
