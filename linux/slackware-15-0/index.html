前言

Slackware 是現存最古老的 Linux 發行版本，今天有幸接觸到這支 Distro。在 DistroWatch 的「網頁參觀排名」，Slackware 排在第 43 位，雖然不是位居前 10 大，但老牌的 Linux 仍有一定的地位。

安裝系統的心得與建議

  1. 習慣近代安裝介面的使用者，可能會比較無法習慣 Slackware，但，對老早就接觸 Linux 的伙伴而言，反而會有一種親切感。
  2. 安裝的過程都是英文，我也沒有特別去看看是不是也支援其他語系介面。
  3. 比較困難的地方是一開始，要先對硬碟做規劃，以我而言，我是從 25GB 的硬碟空間去分割成 3 個磁碟區，分別是 GPT(EFI)：300MB、Swap：1GB，其餘的 23.7GB 切成一個磁碟區(ext4)。
  4. 要進到桌面環境，記得下 startx 這個指令。

安裝嘸蝦米輸入法

  1. 令人非常意外的地方，就是完成系統的預設安裝後， fcitx(4) 與 fcitx-table-boshiamy 都不用再另外安裝。

  2. 但，也不是這樣就可以直接啟用 fcitx(4)，而是先去編輯 /etc/enviroment，以 root 的權限下 vi /etc/enviroment 這道指令，並寫入下列內容(複製貼上當然還是王道)。

  3. 重開機之後，進入系統內的「輸入法」設定，餵關鍵字「boshi」找出「Boshiamy」並且將其加入。

  4. Ctrl+Space 就可以召喚「嘸蝦米輸入法」。

  5. 「嘸蝦米輸入法」完全可以使用，沒有問題。
