Fedora 10 Live USB

發表於 2008-11-30 更新於 2024-03-17 分類於 linux 文章字數： 662 所需閱讀時間 ≈ 1 分鐘

跟前幾個版本的 Fedora 比較，今次我算是玩得晚的乙次…

在 Fedora 9. Make Waves. 的時候，我就想拿隨身碟來玩，在 Fedora 10 釋出前，透過 Fedora LiveUSB Creator 小玩一下下，結果並不如預期。

今兒個在下載 Fedora 10，難得我勤勞地想檢查 ISO 檔，卻一下子找不到 SHA1，所幸 liveusb-creator - Trac 提供此一功能（詳可閱 liveusb-creator - Trac 的 Features 部分）。

今次，我沒有客製化 LiveUSB 的打算，純綷想試試而已，所以直接就拿 Fedora 10 的 ISO 來玩，大致上我試玩了以下幾個部分。

本次的電腦硬體所使用的是 NVIDIA GeForce 7300 GT，之前的經驗所使用的是 SIS 內建的顯卡，當時並沒有進入 X，今次我讓電腦從隨身碟開機後，就去給寶貝們泡牛奶，再洗奶瓶等，回到電腦前，已經在 X 裡頭，索性開個 Desktop Effects，結果是一片空白。

若電腦上沒有二部以上的光碟機，拿 Live CD 開機，要再聽 Music CD 是不可能的事，用 Live USB 開機，我就想來聽聽音樂，不知怎麼搞的，音樂放下去，我就不想重開機，讓音樂放著，也讓連日來的疲累有個出口。

最後，想說的是很可惜在語系的支援部分稍弱一些（或許有支援，在我惟一可能錯過的登入畫面處），而 liveusb-creator - Trac 實在是簡單易用，完全無痛。
