前言

Fedora 40 有什麼樣的更新？

What’s New

從 RedHat 官網讀到這則訊息：Announcing Fedora Linux 40。

我試著把這則訊息裡的標題整理出來，得到下列這些內容。

  1. CentOS Stream 10 & RHEL 10 branch point

  2. Introducing Fedora Atomic Desktops

  3. Bootable IoT containers

  4. AI advancements

  5. Porting to modern C

  6. Individual, stable MAC address assignment for Wi-Fi

  7. Cloud Edition images now built with Kiwi

另外，這則訊息裡面，特別整理 Feature Updates(如下)。

  1. New compilers and tools

  2. KDE Plasma 6

  3. Podman 5

  4. Faster package updates

  5. Python 3.7 retiring

玩轉 1 小時的心得

  2. 不一定要馬上更新：這一點，倒不能全然都說是 Fedora 的問題，除非是想要「新版本號」，不然，如果跟我一樣，只是拿來當桌面系統使用，其實不太需要更新。畢竟，其他功能，我用不到，自然也就碰不到，更不會注意更新與否。
