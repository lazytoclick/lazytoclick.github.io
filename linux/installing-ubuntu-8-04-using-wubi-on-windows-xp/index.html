使用 Windows XP 上的 wubi 安裝 Ubuntu 8.4

發表於 2008-05-08 更新於 2024-03-17 分類於 linux 文章字數： 790 所需閱讀時間 ≈ 1 分鐘

最近，我對 ubuntu 8.04 實在是深深著迷，今兒個即使忙碌到不行，我還是不管任何時間管理上的安排，動起安裝的念頭。

今次使用的是精靈安裝法，精靈名為 wubi (( 全名 Windows based UBuntu Installer ，不是新玩意兒， Debian-Installer Loader 應該算是前輩，另外也有其他 Linux distribution 有類似，甚且更強大的玩具，不過，礙於訊息較為片段、不確定，在此就不多談。 )) ，過程很簡單。

英文的翻釋如下：

  1. Installation Drive：選擇安裝的目標硬碟。
  2. Installation Size：調整安裝的空間。
  3. Desktop Enviroment：直譯為桌面環境，我在安裝時只有 ubuntu 這個選項。
  4. Language：看起來挺帥的，有 Chinese (Traditional)，不過結果並非如此，稍後再述。
  5. Username 與 Password：用戶名與密碼。

點下安裝，緊接著就是安裝進度畫面，很棒的是我注意到過程中會再製作一次光碟的 ISO 檔，以備不時之需。

再來，安裝程式會請使用者重新開機，上述過程至此，我覺得挺有趣的，若不是安裝介面設計得簡單清爽，中文化顯然不足。

最後，我想說的是正體中文並沒有想像般順利的進入介面，呈現出的是簡體中文（漢語），僅管我所遭遇的經驗大部分都是如此，原以為我還得先問一下男人該怎麼使用 apt-get ，未料將 /etc/apt/sources.list 覆蓋，並且把網路設定完之後，點下 Language Support ，選單內就自然出現一大堆語言，我先把介面切換到英語(美國) (( 唉！漢語，我真是不了解你。 )) ，重新開機再進系統，找到並勾選 Chinese 之後，更出乎意料的是語系檔就此自動下載與安裝。
