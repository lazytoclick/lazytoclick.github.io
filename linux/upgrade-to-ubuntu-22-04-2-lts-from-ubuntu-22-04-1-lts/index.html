從 ubuntu 22.04.1 LTS 升級到 ubuntu 22.04.2 LTS

發表於 2023-03-05 更新於 2023-08-03 分類於 linux 文章字數： 817 所需閱讀時間 ≈ 1 分鐘

前言

ubuntu 22.04.2 LTS 這支更新是在 2023 年 2 月 24 日釋出，詳可見 Ubuntu 22.04.2 LTS released。

Linux 上的系統升級方法，我自己是用 CLI(Terminal) 與 GUI(Graphic User Interface) 兩種方式做為區分，而這一篇貼文，主要是在 CLI 裡面，透過指令把虛擬機器裡面的 ubuntu 22.04.1 LTS 升級到 ubuntu 22.04.2 LTS。

升級流程

  1. 我想，先把幾個指令直接寫下來(如下)。

  2. 從下圖，可以看到我是下 sudo apt update && sudo apt upgrade 來做處理。

  3. 我是從全新且最小安裝的 ubuntu 22.04.1 LTS 去做升級的，指令下完之後，跑出「升級 268 個，新安裝 2 個，移除 0 個，有 1 個未被升級。」這樣的訊息(如下圖)，這個部分，依每個人使用系統的情形可能有所不同，一般而言，通常差異也不會太大。如果差異很大的話，也可以從 CLI 裡面去檢視有哪些套件被升級。

  4. 等待指令順順跑完後，再下一次 cat /etc/issue 這一道指令確認一下，沒有問題後，就可以下 reboot 重新開機。

後記

  1. 我在撰寫本篇貼文的時候，ubuntu 在 Distrowatch.com 上「近 6 個月」(Last 6 months)的「網頁參觀排名」是第 7 名。
  2. 至於 ubuntu 22.04.2 LTS 有什麼樣的更新，有興趣的人可以參考官網資料：Jammy Jellyfish Release Notes - Release - Ubuntu Community Hub。
