Red Hat Enterprise Linux 9.4

發表於 2024-06-01 更新於 2024-06-21 分類於 linux 文章字數： 988 所需閱讀時間 ≈ 2 分鐘

前言

今天，來回味一下，多年未再接觸的發行版：Red Hat。

Red Hat Enterprise Linux（RHEL）是一個由Red Hat開發的商業市場導向的Linux發行版。
引用自 Red Hat Enterprise Linux - 維基百科，自由的百科全書

安裝過程

選擇「基礎環境」與「所選環境的額外軟體」

這次是在 VirtualBox 上試用，我想要將系統設定為「工作站」。

啟用 Root 帳號？

我沒有啟用 root 密碼（也就是「Root 帳號已停用」），而是新增一位使用者，並且「讓此使用者成為管理員」。

處理中文輸入法

  1. 選擇以「工作站」做為「基礎環境」的關係，後續處理中文輸入法的部分，就簡單許多。

  2. 直接使用系統預設安裝的 iBus，把官方的嘸蝦米輸入法表格與圖像檔處理一下，就可以搞定，沒有想到，ibus-table 這支套件沒有跟著預設安裝一起裝進系統。所幸，這件事很好解決，下一道指令 sudo dnf install ibus-table，先搞定 ibus-table，再處理嘸蝦米輸入法官方表格與圖像檔。

心得

  1. 在 VirtualBox 上使用圖形介面，容易有小小的卡頓感，這個部分是我目前使用所有的 Linux 發行版中，在安裝之後，就立馬可以感受到的部分。我想，對於想要使用 VirtualBox 來做一些 Lab 的伙伴，多少要顧慮到這一點。

  2. 官方的說明文件非常豐富，遇到使用上的問題，基本上，我會試著先跑一趟 Product Documentation for Red Hat Enterprise Linux 9 | Red Hat Customer Portal，找資源與解法。

  3. 這次的接觸、試用後，我會認為使用 Red Hat Enterprise Linux 需要一些基本功，如果是要管理 Red Hat Enterprise Linux，那麼基本功能夠越紮實是再好不過。
